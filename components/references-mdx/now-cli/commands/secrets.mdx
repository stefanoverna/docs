## Secrets

The `now secrets` command is used to manage Secrets used in [Environment Variables](/docs/v2/build-step#environment-variables) under an account, providing functionality to list, add, rename, and remove Secrets.

### Basic Usage

<Snippet dark text="now secrets list" />
<Caption>Using the <InlineCode>now secrets</InlineCode> command to list all Secrets under an account.</Caption>

### Extended Usage

<Snippet dark text="now secrets add [secret-name] [secret-value]" />
<Caption>Using the <InlineCode>now secrets</InlineCode> command to add a Secret to an account.</Caption>

<Snippet dark text="now secrets rename [old-name] [new-name]" />
<Caption>Using the <InlineCode>now secrets</InlineCode> command to rename a Secret under an account.</Caption>

<Snippet dark text="now secrets remove [secret-name]" />
<Caption>Using the <InlineCode>now secrets</InlineCode> command to remove a Secret from an account.</Caption>

### Global Options

The following [global options](#options) can be passed when using the `now secrets` command:

- `--debug`
- `--global-config`
- `--help`
- `--local-config`
- `--scope`
- `--token`

For more information on global options and their usage, refer to the [options section](#options).
