# Introduction

## Configuration Reference

This page is a reference to the different ways of configuring your <ProductName /> projects and <ProductShortName /> CLI.

Using a `now.json` configuration file, placed at the root of a project, you can provide a list of options that changes the default behavior of the <ProductName /> platform.

Configuration options for the following are described on this page:

| Category                              | Description                                                                                   |
| ------------------------------------- | --------------------------------------------------------------------------------------------- |
| [Project](#project)                   | General configuration options that describe how your project should act with <ProductName />. |
| [Git Integrations](#git-integrations) | Configuration options related to <ProductName /> Git Integrations.                            |
| [Routes](#routes)                     | Routing configuration for projects with <ProductName />.                                      |
| [Global](#global)                     | Configuration options for usage of <ProductShortName /> CLI.                                  |
