## Deploy an Existing Project

Once you have [signed up to <OrgName />](/signup), you can use our [Git Integrations](/docs/v2/git-integrations) with [GitHub](/docs/v2/git-integrations/vercel-for-github), [GitLab](/docs/v2/git-integrations/vercel-for-gitlab), or [Bitbucket](/docs/v2/git-integrations/vercel-for-bitbucket) to deploy an existing project – then with a deployment for every push – by selecting a repository from your Git account.

Select a Git provider below to get started deploying your existing repositories and projects:
